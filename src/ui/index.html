Guia Rápido & Atalhos de Teclado

O CapIAu-Talho é otimizado para navegação ultra-rápida via teclado. Aqui está o resumo de como operar a Timeline Canvas:

2. Comportamento das Trilhas (V1 vs V2)

V1 (Falas): É magnética. Os cortes ficam grudados. Arrastar clipes reordena a sequência de falas; puxar as bordas (trim) ajusta a fala e puxa todo o resto junto.

V2 (B-Roll): É livre. Você pode arrastar os clipes para qualquer ponto no tempo e ajustar a duração sem alterar a fala principal.
